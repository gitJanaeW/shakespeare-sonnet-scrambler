About This Project

This is tool to generate nonsensical Shakespearean poems with no intended purpose other than to practice Javascript randomizers in a structured format.

Sonnets must follow a specific structure, line by line. That means simply randomizing lines is not enough. When randomly assembling this sonnet, the lines must be randomly generated in a prescribed order (ie. starting with the 1st line of a random poem, then the 2nd line of a random poem and so on).

I've started with only Shakespeare's most famous poems, but with time and experience I will add more content and features to this generator.
